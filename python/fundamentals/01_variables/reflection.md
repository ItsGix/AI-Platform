# Week 1 - Module 01 Solution

## What I Learned

- Variables store information.
- Python uses `=` to assign values to variables.
- Variable names should use `snake_case`.
- Python has several common data types:
  - `str` (text)
  - `int` (whole numbers)
  - `float` (decimal numbers)
  - `bool` (True/False)
- `print()` displays information to the terminal.
- `print()` with no arguments prints a blank line.
- `f` before a string creates an f-string.
- Values inside `{}` are replaced with variables.
- `\n` creates a new line inside a string.
