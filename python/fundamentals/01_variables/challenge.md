# Week 1 - Module 01 Challenge

## Objective

Create a Python program that describes your own AI Platform or homelab using variables.

The program should demonstrate your understanding of:

- Variables
- Data Types
- Strings
- Integers
- Floats
- Booleans
- f-Strings
- print()
- Escape characters (\n)

---

## Requirements

Create variables for:

### Gaming PC
- CPU Model
- RAM Size
- GPU Model
- Operating System

### AI Platform
- Cluster Name
- Control Plane Nodes
- Worker Nodes
- GPU Nodes
- Kubernetes Version

### Resource Usage
- CPU Utilization
- RAM Utilization
- GPU Temperature

### Status
- Internet Connected
- Docker Running
- GPU Available
- Cloudflare Tunnel Connected

---

## Challenge

Display all information in a clean and readable format using:

- f-Strings
- New lines (`\n`)
- Section headings
- Blank lines for readability

Example:

=== PC Specs ===
CPU: ...
RAM: ...
GPU: ...

=== Cluster Specs ===
...

---

## Bonus Challenge

Can you make the output look professional enough that it resembles a real command-line tool?

Think about spacing, formatting, and readability.
